.env client 
-----------
VITE_BASE_URL = base url of the server/api (like-localhost:3000/api)


.env backend
----------

EMAIL_USER=email for nodemailer 

EMAIL_PASSWORD= Google App Password for nodemailer

MONGODB_URI= mongodb connecting uri

JWT_SECRET= just a secretkey 

PORT=just a port number for server creation

CLIENT_URL= client url (like-http://localhost:5173/)
